# Reolink-NVR-Manager
Hubitat intergration for making Reolink cameras triger motion events (Person/Vehicle/Motion)
# Reolink NVR Manager for Hubitat

## Features
- Automatic camera discovery
- Motion, Person, and Vehicle sensors
- Local LAN operation only
- No cloud required
- Automatic token renewal

## Requirements
- Hubitat C7/C8
- Reolink NVR with API enabled
- ONVIF/API user account

## Installation
1. Import the app.
2. Import the driver.
3. Install the app.
4. Enter NVR credentials.
5. Click "Discover Cameras".
